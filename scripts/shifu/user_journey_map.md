## Complete If-Then Logic Summary

| Condition | Then |
|-----------|------|
| **Business type ≠ High-Ticket Service** | → Waitlist |
| **Close Rate < 30%** | → "Offer problem. Wrong avatar or sales motion." |
| **Gross Margin < 80%** | → "Offer problem. Commoditized." |
| **Close Rate ≥ 80%** | → "Underpriced 3-4x. Raise price." |
| **Close Rate 60-80%** | → "Underpriced 2-3x. Consider doubling." |
| **Close Rate 30-40%** | → "Appropriately priced. Proceed." |
| **Outreach = 0** | → "Volume problem. Send 20 Looms." |
| **Outreach > 50, Responses = 0** | → "Messaging problem. Review 3 messages." |
| **Responses > 10, Calls = 0** | → "Follow-up problem. Book X calls." |
| **Calls > 5, Closed = 0** | → "Sales problem. Review a call." |
| **Revenue ÷ Clients < $4K** | → "Price problem. Raise to $5K." |
| **Clients > 10, Capacity = 0** | → "Capacity problem. Raise price or drop clients." |
| **Prescription completed** | → "Keep momentum. +10% volume." |
| **First skip** | → "Same prescription." |
| **Second skip** | → "Halved. What would make it possible?" |
| **Third skip** | → "Confrontation moment. Just 5. What's really wrong?" |
| **Same excuse 3x** | → "Pattern surfaced. What would need to change?" |
| **Started but didn't finish** | → Re-engagement reminder |
| **Got prescription, didn't report** | → Follow-up check-in |
| **30+ days since audit** | → Re-audit prompt |
