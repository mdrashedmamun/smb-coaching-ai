# 02 Personal Styling (56 min) - Detailed Analysis

## Snapshot
- Business: Personal styling / wardrobe consulting
- Revenue: ~309,000
- Profit: ~130,000 (42% net margin)
- Leads: ~12/mo, 8 appointments, 4 sales (50% close)
- Core issue stated: lead flow + conversion leak

## Alex's Diagnostic Flow (as observed)
1) Scoreboard: revenue, profit, margin, LTV, CAC, close rate.
2) Avatar clarity to tighten targeting and messaging.
3) Channel mix to identify leverage and re-allocate effort.
4) Funnel leak: clicks -> sales gap caused by missing lead magnet.
5) Pre-sell assets before calls to improve conversion.

## Prescriptions (ordered)
- Narrow the avatar and sharpen the promise.
- Install a high-value lead magnet (risk-free, easy, fast).
- Add pre-call video(s) to reduce drop-off and shorten calls.
- Scale paid ads and affiliates only after funnel improves.
- Build continuity revenue (monthly/quarterly retainer).
